#### 快速上手

```bash

pip install mlx mlx-audio

# 基本使用

# 文本转语音
ma.tts("Hello, this is a test.", "output.wav")

# 语音转文本

# 语音转语音
```

- 此库仅支持Apple Silicon设备(M1/M2/M3芯片)，无法在其他平台上运行
